# Briolette Hardware

Hardware designs for Briolette-compatible devices.

## Projects

### [stick/](stick/)

The "credstick" — a credit-card or USB-stick form factor device with:
- NFC tap-to-pay for anonymous token transfers
- E-ink display showing balance and transaction status
- Hardware secure element for key storage
- USB-C for charging and firmware updates

See [stick/README.md](stick/README.md) for details.
